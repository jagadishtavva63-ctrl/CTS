# Lab 3 — EF Core Migrations

> **Week 2 · CTS Deep Skilling · Entity Framework Core**

## Overview

Demonstrates how to use EF Core migrations to create and update the database schema from C# entity definitions without writing SQL DDL scripts.

### Learning Objectives

- Install and use the `dotnet-ef` CLI tool
- Create an initial migration with `dotnet ef migrations add`
- Apply migrations to SQL Server with `dotnet ef database update`
- Understand the `Migrations/` folder structure
- Reverse a migration with `dotnet ef migrations remove`

## Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
- SQL Server
- EF Core CLI:

```bash
dotnet tool install -g dotnet-ef
```

---

## How to Run

```bash
cd "week 2/Entity Framework Core/Lab3_Migrations"
dotnet restore

# Create migration
dotnet ef migrations add InitialCreate

# Apply to database
dotnet ef database update

# Run the app
dotnet run
```

### Undo a Migration

```bash
dotnet ef migrations remove
```

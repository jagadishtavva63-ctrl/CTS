# jQuery Notes

## Syntax
`$(selector).action()`
Simplifies JS programming.

## Selectors
`$("#id")`
`$(".class")`
Used to find HTML elements.

## Events
// Click event
`$("#btn").click(function(){ ... });`
Handle user interactions.

## Hide and Show
`$("#demo").hide();`
`$("#demo").show();`
Easily toggle visibility of elements.

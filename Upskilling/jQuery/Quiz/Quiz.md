# jQuery Quiz

1. **MCQ**: Which sign is used for jQuery?
   - A) %
   - B) $
   - **Answer**: B

2. **True/False**: jQuery is a server-side language.
   - **Answer**: False

3. **Short Answer**: Which method hides selected elements?
   - **Answer**: `hide()`

4. **MCQ**: How do you select all `<p>` elements?
   - A) `$("p")`
   - B) `$(p)`
   - **Answer**: A

5. **True/False**: `$(this)` refers to the current HTML element.
   - **Answer**: True

6. **Short Answer**: Method to get the value of an input field?
   - **Answer**: `val()`

7. **MCQ**: Method used to set style properties?
   - A) `style()`
   - B) `css()`
   - **Answer**: B

8. **True/False**: `append()` inserts content at the end of the selected elements.
   - **Answer**: True

9. **Short Answer**: Method to perform an asynchronous HTTP request?
   - **Answer**: `$.ajax()`

10. **MCQ**: How to prevent a form from submitting naturally?
    - A) `event.stop()`
    - B) `event.preventDefault()`
    - **Answer**: B

\# CSS3 Basics



\## Introduction

CSS (Cascading Style Sheets) is used to style HTML pages.



\## Advantages

\- Separates content and design

\- Reusable styles

\- Responsive layouts

\- Faster maintenance



\## Types of CSS

1\. Inline CSS

2\. Internal CSS

3\. External CSS



\## CSS Syntax



```css

selector{

&#x20;   property:value;

}

```



Example



```css

h1{

&#x20;   color:blue;

}

```



\## Selectors

# Assignment 2: Simple Blog Post

**Problem Statement**: Create a webpage displaying a blog post using semantic tags.

**Concepts Used**: Semantic Elements (`<article>`, `<header>`, `<section>`).

**Solution**:
```html
<!DOCTYPE html>
<html>
<body>
  <article>
    <header>
      <h1>My First Blog</h1>
      <p>Posted on: Jan 1, 2026</p>
    </header>
    <section>
      <p>This is the main content of my blog post. Learning HTML is fun!</p>
    </section>
    <footer>
      <p>Author: Student</p>
    </footer>
  </article>
</body>
</html>
```

**Expected Output**: A structured article with a header, content area, and footer.

**Short Explanation**: Using `<article>` makes it clear this is independent content. This is better than just using generic `<div>` tags.

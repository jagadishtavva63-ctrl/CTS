# CSS3 Quiz

1. **MCQ**: What does CSS stand for?
   - A) Creative Style Sheets
   - B) Cascading Style Sheets
   - **Answer**: B

2. **True/False**: `padding` is outside the border.
   - **Answer**: False (Margin is outside, Padding is inside).

3. **Short Answer**: How do you make text bold?
   - **Answer**: `font-weight: bold;`

4. **MCQ**: Which property is used to change the background color?
   - A) bgcolor
   - B) background-color
   - **Answer**: B

5. **True/False**: `id` selectors carry more weight than `class` selectors.
   - **Answer**: True

6. **Short Answer**: Which property controls text size?
   - **Answer**: `font-size`

7. **MCQ**: How to remove underline from links?
   - A) `text-decoration: none;`
   - B) `text-style: normal;`
   - **Answer**: A

8. **True/False**: Flexbox is meant for 2D layouts.
   - **Answer**: False (Grid is for 2D, Flexbox is for 1D).

9. **Short Answer**: How to select elements with class "test"?
   - **Answer**: `.test`

10. **MCQ**: Default `position` value of elements?
    - A) relative
    - B) static
    - C) absolute
    - **Answer**: B

11. **True/False**: `rem` is relative to the root html font size.
    - **Answer**: True

12. **Short Answer**: How to apply styles only for screens smaller than 500px?
    - **Answer**: `@media (max-width: 500px)`

13. **MCQ**: Which property creates space between grid items?
    - A) gap
    - B) spacing
    - **Answer**: A

14. **True/False**: `z-index` works on statically positioned elements.
    - **Answer**: False

15. **Short Answer**: Property to make an element transparent?
    - **Answer**: `opacity`

# HTML5 Basics

## Introduction
HTML5 is the latest version of HyperText Markup Language used to create web pages.

## Basic Structure

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>
```
